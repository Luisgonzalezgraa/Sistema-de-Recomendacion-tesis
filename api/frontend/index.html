Acerca de esta API

Este es un servicio RESTful diseñado para realizar análisis multiparamétricos de redes de riego por goteo. La API integra tres módulos especializados que trabajan conjuntamente para proporcionar recomendaciones técnicas completas y fundamentadas en datos geoespaciales y parámetros hidráulicos reales.

Módulos de Análisis

🗺️ Geoespacial

Análisis de terreno mediante DEM, pendientes y detección de zonas críticas para riego.

💧 Hidráulico

Cálculos de presión, caudal y pérdidas de carga usando métodos Hazen-Williams y energía.

⚗️ Recomendación

Motor de decisión inteligente que genera especificaciones de motobomba, tuberías y materiales.

Endpoints Disponibles

Accede a estos puntos de entrada para consultar la documentación, verificar el estado del sistema y procesar análisis completos de tu infraestructura de riego.

GET
/api/v1/health

Verifica el estado actual del servidor y la disponibilidad de la API

GET
/api/v1/docs

Obtiene la documentación completa en formato JSON con descripciones de todos los endpoints

POST
/api/v1/analysis/elevation

Realiza análisis geoespacial: pendientes, elevaciones y zonas críticas a partir de una imagen DEM

POST
/api/v1/analysis/hydraulic

Calcula parámetros hidráulicos: presiones, caudales y pérdidas basado en datos de fuente y terreno

POST
/api/v1/recommendations

Genera recomendaciones técnicas completas: especificaciones de bomba, tuberías y observaciones de diseño

ℹ️ Acerca del Proyecto

Sistema de Recomendación para Redes de Riego por Goteo

Este proyecto es una herramienta desarrollada como seminario de tesis que integra análisis geoespaciales, cálculos hidráulicos y un motor de recomendación inteligente para diseñar redes de riego por goteo optimizadas basadas en parámetros reales del terreno y del agua disponible.

Características Principales

  • 🗺️ Análisis Geoespacial: Procesa modelos de elevación digital para detectar pendientes y zonas críticas
  • 💧 Cálculos Hidráulicos: Determina presiones, caudales y pérdidas de carga usando métodos reconocidos
  • ⚗️ Análisis de Agua: Evalúa compatibilidad del agua con materiales de conducción
  • ⚙️ Recomendaciones Técnicas: Genera especificaciones de bomba, tuberías y materiales
  • 📊 Reportes Detallados: Proporciona documentación completa del diseño recomendado
